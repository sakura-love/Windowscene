# Windows 情景模式 v0.1.0

## 发布说明

Windows 情景模式首个公开版本发布。

这个版本已经具备基础可用能力：

- 开机后弹出“本次开机做什么”情景选择窗口
- 支持游戏、视频、音乐、阅读、上网、编程、工作等情景模式
- 支持按情景推荐应用
- 支持手动配置每个情景的应用
- 支持简单智能识别常用应用
- 支持一键启动所选情景应用
- 支持打包为 Windows EXE

## 界面特性

- 基于 PySide6 构建
- 整体风格偏向 Windows 11
- 主页面左侧分为情景模式与推荐应用两个区域
- 推荐应用按左到右、从上到下排列

## 附件说明

本次 Release 附带：

- `Windowscene.exe`

## 已知限制

- 某些 `.lnk` 或特殊安装方式的软件图标可能无法稳定提取
- 当前软件识别规则仍以关键词匹配为主
- 当前仍属于原型到可用版本之间的阶段
