# Release Template

## Windows 情景模式 v0.1.0

### 新增

- 支持开机后弹出情景选择
- 支持情景模式卡片选择
- 支持推荐应用卡片选择
- 支持手动配置应用
- 支持简单智能识别常用应用
- 支持打包为 Windows EXE

### 使用方式

1. 下载发布包
2. 运行 `Windowscene.exe`
3. 根据自己的使用习惯配置情景应用

### 已知限制

- 某些 `.lnk` 或特殊安装方式的软件图标可能无法稳定提取
- 当前软件识别规则仍以关键词匹配为主

### 后续计划

- 优化 Windows 图标提取
- 增加托盘常驻
- 增加更多系统情景动作
